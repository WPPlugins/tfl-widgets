Add Transport For London (TFL) travel information and/or a Tube map to your pages quickly and easily with these shortcodes.

== Description ==

TFL Widgets lets you add Transport For London widgets to your pages using WordPress "shortcodes". This plugin is not created or endorsed by Transport For London.

Simply install the plugin, then you can add the shortcodes as follows:

#### Tube service updates

Live travel news for the Tube, updated every minute.

* `[tube-updates]` - Gives live Tube service updates 240px wide (default).
* `[tube-updates width="500px"]` - Gives live Tube service updates 500px wide, you can also use any valid CSS width units: %, em etc.
* `[tube-updates width="variable"]` - Gives live Tube service updates of variable width which stretches to fit its container.

#### Tube map

Draggable Tube map, with integrated station search.

* `[tube-map]` - Gives a Tube map 360px wide (default).
* `[tube-map width="500px"]` - Gives a Tube map 500px wide, you can also use any valid CSS width units: %, em etc.
* `[tube-map width="variable"]` - Gives a Tube map of variable width which stretches to fit its container.

== Installation ==

In WordPress 2.7 and above you can install plugins directly from the admin area. Download the plugin to your computer, then log in to your WP admin area and go to Plugins > Add New. Browse to the plugin archive and select it. Then click Install Now and the plugin will be installed shortly.

For more help see [http://www.wpbeginner.com/beginners-guide/step-by-step-guide-to-install-a-wordpress-plugin-for-beginners/](http://www.wpbeginner.com/beginners-guide/step-by-step-guide-to-install-a-wordpress-plugin-for-beginners/ "How to Install a WordPress Plugin - Step by Step for Beginners")
